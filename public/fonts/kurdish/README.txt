Kurdish font: NRT Bold by Ari Mustafa
Source: https://www.kurdfonts.com/ku/font-info/508
License: Free / Open Source
Files in this folder:

NRT-Bd.ttf   (bold, weight 700)
NRT-Reg.ttf  (regular, weight 400)

This font is self-hosted because it is not available on Google Fonts.
It is specifically designed for Kurdish Sorani script and covers the
correct Unicode codepoints that Alexandria does not fully support.
Do not remove these files - they are required for Kurdish UI rendering.
